# Keymaps

`<leader>` = `Space`

---

## 基本操作

| キー | 動作 |
|------|------|
| `qq` | 保存せず全終了 |
| `L` | 行末に移動 |
| `<Esc>` | 検索ハイライト消去 |
| `jk` | インサートモード → ノーマルモード |

---

## モード切替（Vim 標準）

| キー | 動作 |
|------|------|
| `i` | インサートモード |
| `v` | ビジュアルモード |
| `<Esc>` / `<C-c>` | ノーマルモードに戻る |
| `:` | コマンドモード |

---

## ファイル操作

| キー | 動作 |
|------|------|
| `:w` | 保存 |
| `:q` | ウィンドウを閉じる |
| `:wq` | 保存して閉じる |
| `qq` | 保存せず全終了 |

---

## バッファ（タブ）

| キー | 動作 |
|------|------|
| `<S-h>` | 前のバッファ |
| `<S-l>` | 次のバッファ |
| `<leader>q` | 現在のバッファを閉じる（レイアウト維持）|

---

## ウィンドウ移動・分割

| キー | 動作 |
|------|------|
| `<C-←↑↓→>` | ウィンドウ間移動 |
| `<leader>wh/j/k/l` | ウィンドウリサイズ |
| `<leader>w=` | ウィンドウサイズ均等化 |
| `:sp` | 水平分割 |
| `:vsp` | 垂直分割 |

---

## ファイルツリー (neo-tree)

| キー | 動作 |
|------|------|
| `<C-n>` | neo-tree toggle |
| `<leader>e` | neo-tree にフォーカス / エディタに戻る（トグル）|

**ツリー内操作（フォーカス中）:**

| キー | 動作 |
|------|------|
| `Enter` | ファイルを開く / ディレクトリ展開 |
| `a` | ファイル / ディレクトリ作成 |
| `d` | 削除 |
| `r` | リネーム |
| `y` | コピー |
| `x` | カット |
| `p` | ペースト |
| `q` | ツリーを閉じる |
| `R` | 更新（リフレッシュ） |
| `H` | 隠しファイル表示 toggle |

---

## Telescope（ファジーファインダー）

| キー | 動作 |
|------|------|
| `<leader>ff` | ファイル検索 |
| `<leader>fg` | 文字列検索（grep） |
| `<leader>fb` | バッファ一覧 |
| `<leader>fh` | ヘルプ検索 |

**Telescope 内操作:**

| キー | 動作 |
|------|------|
| `<C-j>` / `<C-k>` | 候補を上下移動 |
| `Enter` | 選択して開く |
| `<C-v>` | 垂直分割で開く |
| `<C-x>` | 水平分割で開く |
| `<Esc>` | 閉じる |

---

## ターミナル

| キー | 動作 |
|------|------|
| `<C-\>` | フローティングターミナル toggle |
| `<leader>j` | 下ペインターミナル toggle |
| `<C-\><C-n>` | ターミナル内でノーマルモードへ |

---

## LSP（全ファイル共通）

| キー | 動作 |
|------|------|
| `gd` | 定義へジャンプ |
| `gD` | 宣言へジャンプ |
| `gr` | 参照一覧 |
| `gi` | 実装へジャンプ |
| `K` | ホバードキュメント |
| `<leader>ca` | コードアクション |
| `<leader>rn` | シンボルリネーム |
| `<leader>f` | フォーマット |
| `<leader>cf` | conform でフォーマット |
| `[d` | 前のエラー |
| `]d` | 次のエラー |
| `<leader>d` | エラー詳細フロート |

---

## Rust（.rs ファイルのみ）

| キー | 動作 |
|------|------|
| `<leader>rr` | Runnables |
| `<leader>rt` | Testables |
| `<leader>rd` | Debuggables |
| `<leader>re` | マクロを展開 |
| `<leader>rc` | Cargo.toml を開く |

---

## 診断（Trouble）

| キー | 動作 |
|------|------|
| `<leader>xx` | 全診断パネル toggle |
| `<leader>xX` | 現在バッファの診断 |
| `<leader>xL` | Location list |
| `<leader>xQ` | Quickfix list |

---

## Copilot

| キー | 動作 |
|------|------|
| `Alt+l` | インライン補完を確定 |
| `Alt+w` | 単語単位で確定 |
| `Alt+]` | 次の候補 |
| `Alt+[` | 前の候補 |
| `<C-e>` | 候補を却下 |

---

## 補完（nvim-cmp）

| キー | 動作 |
|------|------|
| `<C-Space>` | 補完を手動トリガー |
| `<Tab>` | 次の候補 / スニペット展開 |
| `<S-Tab>` | 前の候補 |
| `<CR>` | 選択して確定 |
| `<C-e>` | 補完を閉じる |
| `<C-b>` / `<C-f>` | ドキュメントスクロール |

---

## Claude Code

| `Alt+q`      | ウィンドウ表示 / 非表示トグル     |

## Markdown

## Alacritty

|-------------|-------------------------------------------------------------|
| `<C-S-v>`   | クリップボードからペースト（`<C-v>` は nvim 矩形選択に割り当て）|

## WezTerm

| キー | 動作 |
|------|------|
| `<C-S-\>` | ペイン縦分割 |
| `<C-S-->` | ペイン横分割 |
| `<C-S-w>` | 現在のペインを閉じる |
| `<C-S-h/j/k/l>` | WezTerm ペイン間移動 |
| `<C-S-t>` | 新規タブ |
| `<C-Tab>` | 次のタブ |
| `<C-S-Tab>` | 前のタブ |
| `<C-S-v>` | クリップボードからペースト（`<C-v>` は nvim 矩形選択に割り当て）|
